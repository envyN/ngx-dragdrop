Angular component Drag

To make an angular component draggable, use the [ngxDraggable] directive

<my-cmp [ngxDraggable]="true">Drag me!</my-cmp>
Drag me!

Use [ngxDraggedClass] to attach css classes while drag is in progress

<my-cmp [ngxDraggable]="true"
      [ngxDraggedClass]="'draggable'">Drag me!</my-cmp>
Drag me!
