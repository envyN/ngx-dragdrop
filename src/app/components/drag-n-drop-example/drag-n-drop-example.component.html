Drag and Drop

To make an element droppable, use the ngxDroppable directive

(ngxDroppable) is a drop handler and hence you can use it to call the onDrop functionality as needed

<span [ngxDraggable]="true">Drag me!</span>
<div (ngxDroppable)="onDrop($event)">Drop here</div>

Bind to the [ngxDragData] directive to a variable to attach data to the dragged element

<span [ngxDraggable]="true"
        [ngxDragData]="{{'{name:"Foo"}'}}">Drag Foo</span>
<span [ngxDraggable]="true"
        [ngxDragData]="{{'{name:"Bar"}'}}">Drag Bar</span>
<div (ngxDroppable)="onDrop($event)">Dropped {{'\{\{ dropVal \}\}'}}</div>
