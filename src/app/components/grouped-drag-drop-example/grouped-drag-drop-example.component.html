Grouped Drag and Drop

To create groups of draggable and their respective droppable container use the ngxDragTag and ngxDropTags directives

ngxDragTag assigns the given name to the ngxDraggable element and is only droppable on ngxDroppable elements which have the same name in its list of ngxDropTags

All Characters
[ngxDropTags]="['friends', 'got']"
{{char.name}} [ngxDragTag]="'{{char.series}}'"
Game of Thrones Characters
[ngxDropTags]="['got']"
{{char.name}} [ngxDragTag]="'{{char.series}}'"
Friends Characters
[ngxDropTags]="['friends']"
{{char.name}} [ngxDragTag]="'{{char.series}}'"
