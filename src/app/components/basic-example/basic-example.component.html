Basic Drag

To make an element draggable, use the [ngxDraggable] directive

<span [ngxDraggable]="true">Drag me!</span>
Drag me! or this image :

Bind to the [ngxDraggable] directive to a variable to toggle draggability of the element

<span [ngxDraggable]="isDraggable">Drag me!</span>

Use [ngxDraggedClass] to attach css classes while drag is in progress

<span [ngxDraggable]="true"
      [ngxDraggedClass]="'draggable'">Drag me!</span>
